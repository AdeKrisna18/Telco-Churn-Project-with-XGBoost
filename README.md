# Telco Customer Churn Prediction using XGBoost
Proyek ini bertujuan untuk memprediksi probabilitas pelanggan berhenti berlangganan (churn) menggunakan algoritma XGBoost, dengan fokus pada identifikasi faktor risiko utama untuk mendukung strategi retensi bisnis.
# Key Features:
* Exploratory Data Analysis: Menemukan korelasi signifikan antara harga bulanan tinggi ($65-$115) dengan tingkat churn.
* Feature Engineering: Membuat fitur baru TotalServices dan TenureGroup untuk menangkap perilaku pelanggan.
* Data Cleaning: Menangani hidden missing values pada kolom TotalCharges.
* odel Performance: Mencapai skor ROC-AUC 0.915 dan Accuracy 86%.
# Business Insights:
* Pelanggan dengan kontrak Month-to-month memiliki risiko churn tertinggi.
* Biaya bulanan yang tinggi merupakan pemicu utama pelanggan berpindah ke kompetitor
